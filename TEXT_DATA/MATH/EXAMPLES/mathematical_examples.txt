# MATHEMATICAL EXAMPLES - REAL-WORLD APPLICATIONS
# Practical examples showing how mathematics is used in everyday life

# ARITHMETIC EXAMPLES
Shopping: Calculate total cost of items with tax
Example: 3 items at $4.99 each + 8% tax = $4.99 × 3 × 1.08 = $16.17

Cooking: Adjust recipe quantities
Example: Recipe serves 4, need to serve 6: multiply all ingredients by 6/4 = 1.5

Budgeting: Calculate monthly expenses
Example: Rent $1200 + utilities $150 + groceries $400 + transportation $200 = $1950 total

# FRACTION EXAMPLES
Construction: Calculate material needed
Example: Need 3/4 yard of concrete for foundation, each bag makes 1/6 yard: 3/4 ÷ 1/6 = 4.5 bags

Cooking: Convert measurements
Example: Recipe calls for 2/3 cup sugar, only have 1/4 cup measure: 2/3 ÷ 1/4 = 8/3 ≈ 2.67 scoops

# PERCENTAGE EXAMPLES
Sales: Calculate discount price
Example: $80 shirt with 25% discount: $80 × (1 - 0.25) = $60

Interest: Calculate bank account growth
Example: $1000 at 5% annual interest: $1000 × 1.05 = $1050 after one year

Tips: Calculate restaurant tip
Example: $45 bill with 18% tip: $45 × 1.18 = $53.10 total

# ALGEBRA EXAMPLES
Physics: Calculate distance traveled
Example: Car traveling at 60 mph for 2.5 hours: distance = 60 × 2.5 = 150 miles

Business: Calculate break-even point
Example: Fixed costs $5000, variable cost $10 per unit, selling price $25 per unit
Break-even: 5000 + 10x = 25x → 5000 = 15x → x = 333.33 units

Finance: Calculate loan payments
Example: $200,000 loan at 4% interest for 30 years: monthly payment ≈ $955

# GEOMETRY EXAMPLES
Construction: Calculate area of room
Example: Room 12 feet by 15 feet: area = 12 × 15 = 180 square feet

Landscaping: Calculate fence needed
Example: Rectangular yard 50m by 30m: perimeter = 2(50 + 30) = 160m of fence

Art: Calculate canvas size
Example: Square canvas with diagonal 20 inches: side = 20/√2 ≈ 14.14 inches

# TRIGONOMETRY EXAMPLES
Navigation: Calculate distance using triangulation
Example: Two observers 1000m apart see balloon at angles 30° and 45°
Using trigonometry: balloon height ≈ 414m

Architecture: Calculate roof pitch
Example: Roof rises 8 feet over 24 foot span: angle = arctan(8/12) ≈ 33.7°

Surveying: Calculate height of building
Example: From 100m away, angle to top is 40°: height = 100 × tan(40°) ≈ 83.9m

# CALCULUS EXAMPLES
Physics: Calculate velocity from position
Example: Position s(t) = t² - 4t + 3, velocity v(t) = 2t - 4
At t = 3 seconds: v(3) = 2(3) - 4 = 2 m/s

Economics: Calculate marginal cost
Example: Cost function C(q) = q² + 10q + 100, marginal cost = dC/dq = 2q + 10
At q = 50 units: marginal cost = 2(50) + 10 = $110 per unit

Biology: Population growth rate
Example: Population P(t) = 1000e^(0.05t), growth rate = dP/dt = 50e^(0.05t)
At t = 10 years: growth rate = 50e^(0.5) ≈ 82.4 people per year

# PROBABILITY EXAMPLES
Games: Calculate winning chances
Example: Roll two dice, probability of sum 7: 6 combinations out of 36 = 1/6 ≈ 16.67%

Weather: Calculate rain probability
Example: 30% chance of rain today, 40% chance tomorrow
Probability of rain both days = 0.3 × 0.4 = 12%

Insurance: Calculate claim probability
Example: 1 in 1000 drivers have accident per year, probability = 0.001

# STATISTICS EXAMPLES
Sports: Calculate batting average
Example: Baseball player gets 25 hits in 80 at-bats: average = 25/80 = 0.312

Quality Control: Calculate defect rate
Example: 15 defective parts in 1000 produced: defect rate = 15/1000 = 1.5%

Education: Calculate class average
Example: Test scores: 85, 92, 78, 90, 88: average = (85+92+78+90+88)/5 = 86.6

# FINANCE EXAMPLES
Investment: Calculate compound interest
Example: $5000 at 6% compounded annually for 10 years: 5000(1.06)^10 ≈ $8954

Mortgage: Calculate monthly payment
Example: $250,000 loan at 4.5% for 30 years: monthly payment ≈ $1266

Retirement: Calculate retirement savings
Example: Save $500/month at 7% for 30 years: future value ≈ $606,640

# ENGINEERING EXAMPLES
Electrical: Calculate power consumption
Example: Device using 2 amps at 120 volts: power = 2 × 120 = 240 watts

Mechanical: Calculate gear ratio
Example: Input gear 20 teeth, output gear 60 teeth: ratio = 60/20 = 3:1

Civil: Calculate beam load
Example: Steel beam 10m long, load 5000kg/m: total load = 10 × 5000 = 50,000kg

# COMPUTER SCIENCE EXAMPLES
Algorithms: Calculate time complexity
Example: Binary search on 1000 items: worst case comparisons = log₂(1000) ≈ 10

Data Structures: Calculate memory usage
Example: Array of 100 integers: 100 × 4 bytes = 400 bytes

Networks: Calculate data transfer time
Example: 100MB file at 10Mbps: time = 100×8/10 = 80 seconds

# PHYSICS EXAMPLES
Mechanics: Calculate kinetic energy
Example: 1000kg car at 20m/s: KE = 0.5 × 1000 × 20² = 200,000 joules

Thermodynamics: Calculate heat transfer
Example: 2kg water heated from 20°C to 80°C: Q = 2 × 4186 × (80-20) = 502,320 joules

Optics: Calculate lens magnification
Example: Object 10cm from 5cm focal length lens: 1/f = 1/do + 1/di → 1/5 = 1/10 + 1/di → di = 10cm
Magnification = -di/do = -10/10 = -1x

# CHEMISTRY EXAMPLES
Stoichiometry: Calculate product amount
Example: 2H₂ + O₂ → 2H₂O, 4g H₂ produces: (4g/2g/mol) × 18g/mol = 36g H₂O

Solutions: Calculate concentration
Example: 5g salt dissolved in 100mL water: concentration = 5g/0.1L = 50g/L

pH: Calculate from hydrogen ion concentration
Example: [H⁺] = 1×10⁻⁷ M: pH = -log(1×10⁻⁷) = 7

# BIOLOGY EXAMPLES
Population: Calculate growth rate
Example: Bacteria double every hour, starting with 100: after 6 hours = 100 × 2⁶ = 6400

Genetics: Calculate inheritance probability
Example: Cross Aa × Aa: probability of aa offspring = 1/4 = 25%

Pharmacology: Calculate drug dosage
Example: Patient 70kg, dosage 2mg/kg: total dose = 70 × 2 = 140mg

# MEDICAL EXAMPLES
BMI: Calculate body mass index
Example: Height 1.75m, weight 70kg: BMI = 70/(1.75)² = 22.86

Heart Rate: Calculate target heart rate
Example: Age 40, maximum heart rate = 220 - 40 = 180 bpm
Target zone = 180 × 0.7 = 126 bpm

Dosage: Calculate pediatric dose
Example: Adult dose 100mg, child 20kg using Clark's rule: dose = (20/68) × 100 ≈ 29mg

# ENVIRONMENTAL EXAMPLES

Water Usage: Calculate daily consumption

Solar Energy: Calculate panel output
Example: 300W panel, 5 hours of sun: 300 × 5 = 1500 watt-hours = 1.5 kWh

# SPORTS EXAMPLES

# QUANTUM EXAMPLES

# RELATIVITY EXAMPLES

# CHAOS EXAMPLES
