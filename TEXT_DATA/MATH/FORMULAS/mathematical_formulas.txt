# MATHEMATICAL FORMULAS - COMPREHENSIVE REFERENCE
# Essential formulas for all areas of mathematics

# ARITHMETIC FORMULAS
Addition: a + b = sum
Subtraction: a - b = difference
Multiplication: a × b = product
Division: a ÷ b = quotient

# FRACTION FORMULAS
Addition: a/b + c/d = (ad + bc)/bd
Subtraction: a/b - c/d = (ad - bc)/bd
Multiplication: a/b × c/d = ac/bd
Division: (a/b) ÷ (c/d) = ad/bc

# PERCENTAGE FORMULAS
Percentage: (part/whole) × 100%
Percent of number: number × (percent/100)
Original price: sale price ÷ (1 - discount rate)
Tax amount: price × tax rate

# EXPONENT FORMULAS
Product of powers: a^m × a^n = a^(m+n)
Quotient of powers: a^m ÷ a^n = a^(m-n)
Power of power: (a^m)^n = a^(mn)
Power of product: (ab)^n = a^n × b^n
Power of quotient: (a/b)^n = a^n/b^n
Negative exponent: a^(-n) = 1/a^n
Zero exponent: a^0 = 1 (a ≠ 0)
Fractional exponent: a^(m/n) = ⁿ√(a^m)

# ROOT FORMULAS
Square root: √a × √a = a
Product of roots: √(ab) = √a × √b
Quotient of roots: √(a/b) = √a/√b
Root of root: ⁿ√(√a) = ²√(ⁿ√a)

# ALGEBRA FORMULAS
Quadratic formula: x = [-b ± √(b² - 4ac)]/(2a)
Discriminant: Δ = b² - 4ac
Sum of roots: -b/a
Product of roots: c/a

# FACTORING FORMULAS
Difference of squares: a² - b² = (a - b)(a + b)
Sum of squares: a² + b² = (a + bi)(a - bi)
Difference of cubes: a³ - b³ = (a - b)(a² + ab + b²)
Sum of cubes: a³ + b³ = (a + b)(a² - ab + b²)
Perfect square trinomial: a² + 2ab + b² = (a + b)²
Perfect square trinomial: a² - 2ab + b² = (a - b)²

# LINEAR EQUATIONS
Slope-intercept form: y = mx + b
Point-slope form: y - y₁ = m(x - x₁)
Standard form: Ax + By = C
Slope formula: m = (y₂ - y₁)/(x₂ - x₁)
Distance formula: d = √[(x₂ - x₁)² + (y₂ - y₁)²]
Midpoint formula: ((x₁ + x₂)/2, (y₁ + y₂)/2)

# SYSTEMS OF EQUATIONS
Two-variable system:
  Substitution method
  Elimination method
  Matrix method: AX = B, X = A⁻¹B

# INEQUALITIES
Linear inequality: ax + b > c
Quadratic inequality: ax² + bx + c > 0
Absolute value inequality: |x| < a means -a < x < a

# LOGARITHM FORMULAS
Product rule: log(ab) = log(a) + log(b)
Quotient rule: log(a/b) = log(a) - log(b)
Power rule: log(a^n) = n log(a)
Change of base: log_b(a) = log_c(a)/log_c(b)
Natural log: ln(a) = log_e(a)
Common log: log(a) = log_10(a)

# EXPONENTIAL FORMULAS
Exponential growth: A = P(1 + r)^t
Exponential decay: A = P(1 - r)^t
Continuous growth: A = Pe^(rt)
Compound interest: A = P(1 + r/n)^(nt)

# SEQUENCE FORMULAS
Arithmetic sequence: a_n = a₁ + (n - 1)d
Arithmetic series: S_n = n/2(a₁ + a_n) = n/2[2a₁ + (n - 1)d]
Geometric sequence: a_n = a₁ × r^(n - 1)
Geometric series: S_n = a₁(1 - r^n)/(1 - r) (r ≠ 1)
Infinite geometric series: S = a₁/(1 - r) (|r| < 1)

# GEOMETRY FORMULAS - 2D
Rectangle:
  Perimeter: P = 2(l + w)
  Area: A = lw
  Diagonal: d = √(l² + w²)

Square:
  Perimeter: P = 4s
  Area: A = s²
  Diagonal: d = s√2

Triangle:
  Perimeter: P = a + b + c
  Area: A = (1/2)bh
  Heron's formula: A = √[s(s - a)(s - b)(s - c)] where s = (a + b + c)/2
  Pythagorean theorem: a² + b² = c²

Circle:
  Circumference: C = 2πr = πd
  Area: A = πr²
  Arc length: L = (θ/360°) × 2πr
  Sector area: A = (θ/360°) × πr²

Trapezoid:
  Area: A = (1/2)(b₁ + b₂)h
  Perimeter: P = a + b₁ + b₂ + c

Parallelogram:
  Area: A = bh
  Perimeter: P = 2(a + b)

# GEOMETRY FORMULAS - 3D
Rectangular prism:
  Volume: V = lwh
  Surface area: SA = 2(lw + lh + wh)
  Space diagonal: d = √(l² + w² + h²)

Cube:
  Volume: V = s³
  Surface area: SA = 6s²
  Space diagonal: d = s√3

Cylinder:
  Volume: V = πr²h
  Surface area: SA = 2πr² + 2πrh
  Lateral surface area: LSA = 2πrh

Sphere:
  Volume: V = (4/3)πr³
  Surface area: SA = 4πr²

Cone:
  Volume: V = (1/3)πr²h
  Surface area: SA = πr² + πr√(r² + h²)
  Lateral surface area: LSA = πr√(r² + h²)

Pyramid:
  Volume: V = (1/3)Bh where B is base area
  Surface area: SA = B + (1/2)Pl where P is perimeter of base, l is slant height

# TRIGONOMETRY FORMULAS
Right triangle:
  sin(θ) = opposite/hypotenuse
  cos(θ) = adjacent/hypotenuse
  tan(θ) = opposite/adjacent
  csc(θ) = hypotenuse/opposite
  sec(θ) = hypotenuse/adjacent
  cot(θ) = adjacent/opposite

Unit circle:
  sin²(θ) + cos²(θ) = 1
  tan(θ) = sin(θ)/cos(θ)
  1 + tan²(θ) = sec²(θ)
  1 + cot²(θ) = csc²(θ)

Angle formulas:
  sin(α ± β) = sin(α)cos(β) ± cos(α)sin(β)
  cos(α ± β) = cos(α)cos(β) ∓ sin(α)sin(β)
  tan(α ± β) = (tan(α) ± tan(β))/(1 ∓ tan(α)tan(β))

Double angle:
  sin(2θ) = 2sin(θ)cos(θ)
  cos(2θ) = cos²(θ) - sin²(θ) = 2cos²(θ) - 1 = 1 - 2sin²(θ)
  tan(2θ) = 2tan(θ)/(1 - tan²(θ))

Half angle:
  sin(θ/2) = ±√[(1 - cos(θ))/2]
  cos(θ/2) = ±√[(1 + cos(θ))/2]
  tan(θ/2) = ±√[(1 - cos(θ))/(1 + cos(θ))] = sin(θ)/(1 + cos(θ))

Law of sines: a/sin(α) = b/sin(β) = c/sin(γ)
Law of cosines: c² = a² + b² - 2ab cos(γ)
Area of triangle: A = (1/2)ab sin(γ)

# CALCULUS FORMULAS - LIMITS
Limit properties:
  lim(f(x) + g(x)) = lim(f(x)) + lim(g(x))
  lim(f(x) - g(x)) = lim(f(x)) - lim(g(x))
  lim(f(x) × g(x)) = lim(f(x)) × lim(g(x))
  lim(f(x)/g(x)) = lim(f(x))/lim(g(x)) if lim(g(x)) ≠ 0

Special limits:
  lim(x→0) sin(x)/x = 1
  lim(x→0) (1 - cos(x))/x = 0
  lim(x→∞) (1 + 1/x)^x = e
  lim(x→0) (1 + x)^(1/x) = e

# CALCULUS FORMULAS - DERIVATIVES
Basic derivatives:
  d/dx(c) = 0
  d/dx(x^n) = nx^(n-1)
  d/dx(e^x) = e^x
  d/dx(a^x) = a^x ln(a)
  d/dx(ln(x)) = 1/x
  d/dx(log_a(x)) = 1/(x ln(a))

Trigonometric derivatives:
  d/dx(sin(x)) = cos(x)
  d/dx(cos(x)) = -sin(x)
  d/dx(tan(x)) = sec²(x)
  d/dx(cot(x)) = -csc²(x)
  d/dx(sec(x)) = sec(x)tan(x)
  d/dx(csc(x)) = -csc(x)cot(x)

Inverse trigonometric derivatives:
  d/dx(arcsin(x)) = 1/√(1 - x²)
  d/dx(arccos(x)) = -1/√(1 - x²)
  d/dx(arctan(x)) = 1/(1 + x²)

Derivative rules:
  Product rule: (fg)' = f'g + fg'
  Quotient rule: (f/g)' = (f'g - fg')/g²
  Chain rule: (f(g(x)))' = f'(g(x)) × g'(x)
  Implicit differentiation: differentiate both sides and solve for dy/dx

# CALCULUS FORMULAS - INTEGRALS
Basic integrals:
  ∫c dx = cx + C
  ∫x^n dx = x^(n+1)/(n+1) + C (n ≠ -1)
  ∫1/x dx = ln|x| + C
  ∫e^x dx = e^x + C
  ∫a^x dx = a^x/ln(a) + C

Trigonometric integrals:
  ∫sin(x) dx = -cos(x) + C
  ∫cos(x) dx = sin(x) + C
  ∫sec²(x) dx = tan(x) + C
  ∫csc²(x) dx = -cot(x) + C
  ∫sec(x)tan(x) dx = sec(x) + C
  ∫csc(x)cot(x) dx = -csc(x) + C

Integration techniques:
  Integration by parts: ∫udv = uv - ∫vdu
  Substitution: u-substitution method
  Partial fractions: decompose rational functions
  Trigonometric substitution: use trig identities

# CALCULUS FORMULAS - APPLICATIONS
Area under curve: A = ∫[a to b] f(x)dx
Volume of revolution: V = π∫[a to b] [f(x)]²dx
Arc length: L = ∫[a to b] √[1 + (f'(x))²]dx
Surface area: SA = 2π∫[a to b] f(x)√[1 + (f'(x))²]dx
Work: W = ∫F(x)dx
Average value: f_avg = (1/(b - a))∫[a to b] f(x)dx

# DIFFERENTIAL EQUATIONS
First-order linear: dy/dx + P(x)y = Q(x)
Separation of variables: dy/dx = f(x)g(y)
Homogeneous: dy/dx = f(y/x)
Exact: M(x,y)dx + N(x,y)dy = 0

# PROBABILITY FORMULAS
Basic probability: P(A) = number of favorable outcomes/total outcomes
Complement: P(A') = 1 - P(A)
Union: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
Intersection: P(A ∩ B) = P(A) × P(B|A)
Conditional probability: P(A|B) = P(A ∩ B)/P(B)

Combinations: C(n,k) = n!/(k!(n - k)!)
Permutations: P(n,k) = n!/(n - k)!
Factorial: n! = n × (n - 1) × ... × 2 × 1

Expected value: E(X) = Σ[x × P(x)]
Variance: Var(X) = E(X²) - [E(X)]²
Standard deviation: σ = √Var(X)

# STATISTICS FORMULAS
Mean: μ = Σx/n
Median: middle value when ordered
Mode: most frequent value
Range: max - min
Interquartile range: Q3 - Q1
Variance: σ² = Σ(x - μ)²/n
Standard deviation: σ = √[Σ(x - μ)²/n]

# LINEAR ALGEBRA FORMULAS
Matrix operations:
  Addition: A + B = [a_ij + b_ij]
  Multiplication: AB = [Σa_ik × b_kj]
  Transpose: A^T where A^T_ij = A_ji
  Inverse: A⁻¹ where AA⁻¹ = I
  Determinant: det(A) for 2×2: ad - bc

Vector operations:
  Dot product: a · b = Σa_i × b_i
  Cross product: a × b = vector perpendicular to both
  Magnitude: |a| = √(a · a)
  Angle: cos(θ) = (a · b)/(|a||b|)

# NUMBER THEORY FORMULAS
Divisibility: a divides b if b = ka for some integer k
Greatest common divisor: gcd(a,b)
Least common multiple: lcm(a,b) = |ab|/gcd(a,b)
Prime number: p > 1 with only divisors 1 and p
Fermat's little theorem: a^(p-1) ≡ 1 (mod p) for prime p
Euler's theorem: a^φ(n) ≡ 1 (mod n) where gcd(a,n) = 1

# COMPLEX NUMBERS
Basic form: z = a + bi
Polar form: z = r(cos(θ) + i sin(θ))
Exponential form: z = re^(iθ)
Modulus: |z| = √(a² + b²)
Argument: arg(z) = arctan(b/a)
De Moivre's theorem: [r(cos(θ) + i sin(θ))]^n = r^n(cos(nθ) + i sin(nθ))

# SERIES FORMULAS
Arithmetic series: S_n = n/2(2a₁ + (n - 1)d)
Geometric series: S_n = a₁(1 - r^n)/(1 - r)
Power series: Σa_n(x - c)^n
Taylor series: f(x) = Σ[f^(n)(c)/n!](x - c)^n
Maclaurin series: f(x) = Σ[f^(n)(0)/n!]x^n

# FOURIER SERIES
Fourier series: f(x) = a₀/2 + Σ[a_n cos(nπx/L) + b_n sin(nπx/L)]
Coefficients:
  a₀ = (1/L)∫[-L to L] f(x)dx
  a_n = (1/L)∫[-L to L] f(x)cos(nπx/L)dx
  b_n = (1/L)∫[-L to L] f(x)sin(nπx/L)dx

# LAPLACE TRANSFORMS
Laplace transform: L{f(t)} = ∫[0 to ∞] e^(-st)f(t)dt
Common transforms:
  L{1} = 1/s
  L{t^n} = n!/s^(n+1)
  L{e^(at)} = 1/(s - a)
  L{sin(at)} = a/(s² + a²)
  L{cos(at)} = s/(s² + a²)

# DISCRETE MATHEMATICS
Recurrence relations: a_n = f(a_(n-1), a_(n-2), ...)
Fibonacci sequence: F_n = F_(n-1) + F_(n-2)
Binomial theorem: (a + b)^n = Σ[C(n,k)a^(n-k)b^k]
Pigeonhole principle: if n items in m containers and n > m, at least one container has >1 item

# OPTIMIZATION FORMULAS
Linear programming: maximize/minimize objective function subject to constraints
Simplex method: algorithm for solving linear programming problems
Lagrange multipliers: ∇f = λ∇g for constrained optimization

# NUMERICAL METHODS
Newton's method: x_(n+1) = x_n - f(x_n)/f'(x_n)
Bisection method: root finding by interval halving
Secant method: x_(n+1) = x_n - f(x_n)(x_n - x_(n-1))/(f(x_n) - f(x_(n-1)))
Trapezoidal rule: ∫[a to b] f(x)dx ≈ (b - a)/2[f(a) + f(b)]
Simpson's rule: ∫[a to b] f(x)dx ≈ (b - a)/6[f(a) + 4f((a + b)/2) + f(b)]

# PHYSICS FORMULAS
Kinematics:
  v = v₀ + at
  x = x₀ + v₀t + (1/2)at²
  v² = v₀² + 2a(x - x₀)

Dynamics:
  F = ma
  F = G(m₁m₂)/r²
  p = mv
  KE = (1/2)mv²
  PE = mgh
  Work: W = Fd cos(θ)
  Power: P = W/t = Fv

Waves:
  v = fλ
  f = 1/T
  ω = 2πf
  k = 2π/λ

Electricity:
  V = IR
  P = IV = I²R = V²/R
  C = Q/V
  E = V/d
  R = ρL/A

# CHEMISTRY FORMULAS
Ideal gas law: PV = nRT
Concentration: c = n/V
pH: pH = -log[H⁺]
pOH: pOH = -log[OH⁻]
pKw: pKw = pH + pOH = 14
Molarity: M = moles of solute/liters of solution
Molality: m = moles of solute/kilograms of solvent

# ECONOMICS FORMULAS
Supply and demand: equilibrium where supply = demand
Elasticity: E = (% change in quantity)/(% change in price)
Compound interest: A = P(1 + r/n)^(nt)
Present value: PV = FV/(1 + r)^t
Annuity: PV = PMT × [(1 - (1 + r)^(-n))/r]

# ENGINEERING FORMULAS
Stress: σ = F/A
Strain: ε = ΔL/L
Young's modulus: E = σ/ε
Safety factor: SF = Ultimate strength/Working stress
Efficiency: η = Output/Input × 100%
Power transmission: P = Tω where T is torque, ω is angular velocity
